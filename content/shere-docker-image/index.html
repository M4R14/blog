วิธีแบ่งปั่น Dockerfile ให้ชาวบ้านใช้ กันบ้าง

วิธีแบ่งปั่น Dockerfile ให้ชาวบ้าน ก็คือการที่เราเขียน Dockerfile จะส่งให้คนอื่นใช่โดนที่ให้เพื่อนDav ของคุณเอาไปใช้เพียงแค่
$ docker pull your_name/your_dockerfile

มาเริ่นกันเริ่ม
  1. Log in Docker Cloud ด้วยคำสั่ง “docker login” ถ้าใครยังไม่มี Account ก็ไปสมัครส่ะ Link อยู่นี่ล่ะ https://hub.docker.com/
    $ docker login
  2. Tag Docker image ที่สร้าง ด้วยคำสั่ง “docker tag” คือจะรันคำสั่งนี้ ก็ต้อง build Dockerfile ก่อนอ่ะ (my_image นี่เราตั้งขึ้นเอง อยากให้ชื่ออะไร ก็ตั้งไป)
    $ docker tag my_image docker_id/my_image
  3. ถึงตรงนี้คือส่ง Docker image ให้ชาวบ้านใช้ล่ะ คำสั่งคือ “docker push”
    $ docker push docker_id/my_image
เพียงเท่านี้ ชาวบ้าน ชาวช่อง ก็จะใช้ Docker image ที่เราทำไว้ได้ล่ะ
